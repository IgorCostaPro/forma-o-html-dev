Agora vamos explicar sobre as tags: hr e a. a tag hr cria uma linha para dividir textos. Ex: No título lá em cima pode ver que embaixo tem uma linha divisória.

A tag a, faz uma ancoragem em algum link externo ou até mesmo linkando dentro da página, como se fosse menu. Ex: Quando lá em cima foi utlizado os menus para vir até o texto

Utilizar novas tags sugeridas: font, del, p, abbr (a ideia é buscar estas tags na internet, entender como ela funciona e utilizar no texto).

De acordo com o w3schools, site recomendado pelo professor. A font tag foi usada no HTML 4 para especificar o tipo da fonte, o tamanho da fonte e a cor do texto.

Sendo assim, no HTML 5, ela foi descontinuada.

A tag del, funciona de uma forma muito legal. Ela passa um traço em algum elemento e correge corrige

A tag abbr define uma abreviatura ou um acrônimo, como HTML. E ai vai uma dica: se usarmos o atributo title conseguimos fazer uma caixinha na abreviação para quando a pessoa passar o mouse em cima.
